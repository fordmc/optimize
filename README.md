This repository contains standardized optimization profiles for Minecraft servers.

## Profiles Available
- **Normal Optimize**: Balanced settings for typical servers. (See `normal_optimize.yml`)
- **Advanced Optimize**: Aggressive settings for larger servers requiring better performance. (See `advanced_optimize.yml`)
- **Extreme Optimize**: Maximum performance tier. (See `extreme_optimize.yml`)

These settings can be manually applied to `server.properties`, `bukkit.yml`, `spigot.yml`, `paper-world-defaults.yml`, and `purpur.yml`. 

Alternatively, if you are using the FordMC pterodactyl panel, you can instantly apply optimal targets through the **Optimizer** tab in your server dashboard.
